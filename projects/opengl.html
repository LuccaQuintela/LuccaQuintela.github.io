Overview

This project is simply a visual demonstration of a pirate ship built entirely in C code to plot points and apply textures. Users can change perspective options and the time of day. During day time, there is an orbiting "sun-like" light source up in the sky that revolves around the ship, or you can make it night time and the scene relies on set torches for light. The ocean moves dynamically with time, both in terms of its shape and its texture, giving it a more realistic appearance. The ship takes the movement of the ocean into account, calculating a derivative between points in the waves. Then with these angles I make the boat respond accordingly to the waves, rotating about all 3 axes. I used bezier curves to create the ropes connecting the different parts of the masts as well.
